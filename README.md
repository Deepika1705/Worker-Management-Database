**Worker Management System**
**Project Overview**

Developed a database management system using MySQL to manage worker information, attendance, roles, and salary details.

**Features**
Add worker details
Update worker information
Track attendance
Manage salary records
Retrieve worker data using SQL queries
Generate reports using joins and aggregate functions
