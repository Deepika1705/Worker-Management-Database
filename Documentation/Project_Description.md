**Project Description**

Worker Management System is a database management project developed using MySQL to efficiently manage worker information, attendance records, and salary details. The system enables storing, updating, retrieving, and managing worker data through SQL queries. It demonstrates the use of database design concepts, table relationships, constraints, joins, and aggregate functions to ensure accurate data management and reporting.

**Features**
Manage worker details and job information
Track attendance records
Store salary information
Perform CRUD operations (Create, Read, Update, Delete)
Generate reports using SQL queries
Maintain data integrity using primary and foreign keys

**Technologies Used**
MySQL
SQL
**SQL Concepts Implemented**
DDL Commands (CREATE, ALTER, DROP)
DML Commands (INSERT, UPDATE, DELETE)
Joins
Constraints
Aggregate Functions
Subqueries
**Project Objective**

To design and implement a relational database system that simplifies worker data management and demonstrates practical SQL skills such as database creation, data manipulation, and report generation.
